# ColHelper
Geometry Dash 합작하는 사람들을 위한 툴.

### 사용방법
1. **Geometry Dash를 종료합니다**
1. `main.py` 또는 `ColHelper.exe`를 실행하세요
2. 세 가지 기능 중 선택하세요
    1. 맵 선택
    2. 세이브 복원(미완성)
    3. 프로그램 종료
3. 선택하고자 하는 맵을 **숫자로** 적어주세요
4. 선택할 Col 범위의 시작을 선택해주세요
5. 선택할 Col 범위의 끝을 정해주세요
6. 자동으로 Col을 오브젝트로 추출해주고 `(0,15)` 위치에 배치합니다
